IDI

Statistics New Zealand Integrated Data Infrastructure

View on GitHub

This repository provides examples of code created for use in the IDI.

If you are a current or potential IDI user who wishes to access more IDI-related code, you can join MeetaData, an online collaborative space created by Statistics NZ for all microdata users and suppliers. Email meetadata@stats.govt.nz.
